# Week 4 Backend Assignment

## Name
Ezekiel

## Description
Todo API built with Node.js HTTP module without Express.js.

## Features
- GET /
- GET /tasks
- POST /add-task
- DELETE /delete-task

## How to Run

```bash
npm install
node server.js
```

## Test Endpoints

### Get tasks
GET http://localhost:3000/tasks

### Add task
POST http://localhost:3000/add-task?task=Exercise

### Delete task
DELETE http://localhost:3000/delete-task?task=Read
